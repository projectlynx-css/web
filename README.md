# 🐆 Project Lynx — Official CS Society Website

Official website of **Project Lynx**, the Computer Science Society of Al-Kawthar University.

## 🚀 Pages

| Page | File | Description |
|------|------|-------------|
| Home | `index.html` | Hero, About, Vision/Mission, Event, Domains, Projects, Stats |
| Timeline | `timeline.html` | Full event & seminar history |
| Projects | `projects.html` | Student project showcase |
| Our Team | `team.html` | Executive committee profiles |
| Hall of Fame | `hall-of-fame.html` | Competition winners & achievers |
| Newsletter | `newsletter.html` | Coming soon terminal page |
| Contact | `contact.html` | Contact form + FAQ |

## 🎨 Design System

- **Primary:** `#00D4FF`
- **Secondary:** `#7B61FF`
- **Accent:** `#00FFC8`
- **Background:** `#050816`
- **Font:** Inter + Poppins (Google Fonts)

## ⚡ Features

- Animated particle background
- Glassmorphism cards
- Live countdown timer (AI & Cybersecurity Summit 2026)
- Scroll reveal animations
- Counter animations
- Cyberpunk terminal (Newsletter)
- 3D tilt hover effects on cards
- Mobile-responsive navigation
- SEO meta tags + Open Graph

## 🌐 Deployment

Deploy to GitHub Pages:
1. Push this folder to a GitHub repo
2. Go to Settings → Pages → Branch: `main` / `root`
3. Done — live at `https://yourusername.github.io/project-lynx/`
